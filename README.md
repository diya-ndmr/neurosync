# Neurosync

## Overview
Neurosync is a University Intelligence Platform built using Microsoft Fabric.

The platform helps universities identify at-risk students by analysing attendance, wellbeing, study habits, academic performance, and intervention data.

## Key Features
- Student Wellbeing Analytics
- Attendance Risk Analysis
- Academic Risk Analysis
- AI Insight Agent
- Intervention Recommendations
- Executive Dashboards

## Architecture

Student Data
↓
Microsoft Fabric Lakehouse
↓
PySpark Notebook
↓
Intelligence Tables
↓
Semantic Model
↓
Dashboard & AI Agent
